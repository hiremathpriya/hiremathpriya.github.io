In the JavaScript language, describe all the use cases for each of: parentheses () - e.g. '1. Execute a function var result = doSomething()' also to surround conditional statements. e.g if() { };

brackets [] Are mostly used for accessing the properties of an object or the element of an Array. e.g var myFruits = ['Mango', 'Banana', 'Apple']; alert myFruits[2]; will get the the third element in the Array.

braces {} Are used during the declaration of object literals. for eg. var myFruits = { Banana: 'Yellow', Apple: 'Red', } etc. it is also used to enclose the block of code like in function definations,conditional blocks, and loops etc. for eg. function --() = { }; then for the loop, for() { }; and if statements if () { };

single quotes '' we use single quotes when we are handling the strings in JavaScript. You'll want to use single quotes where you want double quotes to appear inside the string (e.g. for html attributes) without having to escape them, or vice versa. for eg. var myName = 'Mini';

double quotes "" You'll want to use single quotes where you want double quotes to appear inside the string (e.g. for html attributes) without having to escape them, or vice versa. var myCar = 'My car is "RED" ' console.log(myCar); My car is "RED"
