Explain the difference between accessing data from arrays and objects.

Arrays and object literals can contain items. The difference between them is how those items are stored. With arrays they will just be a list of items: 20, willis street, salesman, however, with object literals we can go a step further to give ID's to each of the object items: Age: 20, Area of work: willis St, Ocupation: salesman. They also differ in how we access those values. With Object literals, you access their values using dot notation (console.log(person.location);), with arrays, you access the values using numerical index (console.log(array[2]);)

Exlain what functions are and why they are useful.

In Javascript a function is a block of code that is designed to perform a particular task. These are executed when something calls that function. For Example, you could set up a key press to trigger a function, that function could do anything, from move a player in a game forward, to submit a for or change a website to a mobile view.
